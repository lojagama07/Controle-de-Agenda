**Welcome to your Supabase project**

This project is configured to run locally using Supabase for authentication and data storage.

**Quick start**

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set these environment variables:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_anon_key
```

Run the app: `npm run dev`

**Notes**

- The app uses Supabase for auth and backend data.
